## 下載 & 使用 
```bash
git clone https://github.com/peter910820/KafuChino.git
cd ./KafuChino/
pip install -r requirements.txt
```  
根據 **.env.example** 檔案填入相關參數  
```bash
python main.py
```  
也可以使用nohup讓他在背景執行:  
```bash
nohup python main.py &
```
